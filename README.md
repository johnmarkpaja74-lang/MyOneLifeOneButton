# **Reflection**




## **What was your game idea?**
Bakadongs is a one-button rhythm game where players tap to lock an orbiting signal dot into target zones. Features multiple zone types: main (scoring), bonus (slow + widen), debuff (speed + shrink), and multiplier (up to 4x).

## **What was the most difficult part to implement?**
The smooth game loop using requestAnimationFrame with delta-time calculations for frame-rate independent animation. Coordinating audio + haptics while maintaining responsive touch input was also challenging.

## **What would you improve with more time?**
-Progression system with difficulty levels and unlockable themes

-Particle effects and smoother transitions

-Global leaderboard using Expo's backend

-Collectible power-ups for more strategy

-Adaptive background music

-Tutorial levels for new players
